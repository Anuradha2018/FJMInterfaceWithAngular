FJM API consumption in Angular

FJM API is a service for monitoring the status of jobs submitted to our supercomputer.

The base url for the first version of the api is http://localhost:5000/about, next you have the endpoints: About, Jobs, Job/JobID, and Stats. All requests has been made with HTTP GET requests
